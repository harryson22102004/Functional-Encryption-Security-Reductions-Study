# Functional Encryption & Security Reductions
Advanced cryptographic systems that allow selective computation on encrypted data.

### 🔬 Research Highlights
- **Predicate Encryption**: Implements logic where a secret key allows a user to learn $f(x)$ from an encryption of $x$, and nothing else.
- **Security Reductions**: Provides a theoretical framework for reducing the security of the scheme to hard problems like LWE or CDH.
- **Cryptography Focus**: A deep dive into privacy-preserving computations.

**Tech Stack:** Python, `Charm-Crypto`.
